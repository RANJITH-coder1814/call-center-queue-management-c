# call-center-queue-management-c
A simple Call Center Help Desk simulation using Circular Queue in C. This project demonstrates queue operations such as enqueue, dequeue, display, and statistics using array-based implementation.

This project simulates a **Call Center Help Desk System** using **Circular Queue implementation in C**. It manages incoming calls, assigns ticket numbers, and serves calls in FIFO order.
